  Q: Will I lose my call history?
  A: No! "Recycle" keeps all history. Only "Unassign All" 
     removes current assignments (but keeps the numbers).

  Q: Which option should I use?
  A: Start with OPTION A (Recycle). It's the safest.
     Only use OPTION B if you want to start completely fresh.

  Q: Can I undo this?
  A: OPTION A and C are safe (can't really "undo" but no harm).
     OPTION B removes assignments - you'll need to reassign.

  Q: What if "Recycle" finds 0 completed calls?
  A: Use OPTION B or OPTION C instead. Your agents haven't 
     called any numbers yet, so there's nothing to recycle.
